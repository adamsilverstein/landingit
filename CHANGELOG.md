# Changelog

## [1.0.0] - 2026-04-01

### Added
- Lightweight, keyboard-driven GitHub PR dashboard built with React and TypeScript.
- Real-time CI status and review state tracking.
- Vim-style keyboard navigation.
- Multi-repository support with configurable repo list.
- PR filtering by state (draft, open, merged), ownership, labels, and search.
- Merge-ready PR highlighting.
- Milestone grouping support.
- Dark and light theme support with keyboard shortcut (`t`).
- Detail panel with PR preview (diff stats, labels, reviewers, check runs).
- Column visibility and reorder customization.
- Auto-refresh with configurable interval.
- Desktop app support via Electron (macOS, Linux, Windows).
- CLI support via `npx landingit` (previously `npx git-dashboard`).
- Rate limit monitoring in the header.
- Unseen PR indicators.
